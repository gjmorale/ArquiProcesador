Para el desarrollo de la aplicación web, elegimos como entorno de desarrollo
el framework Node.js. Node.js es un framework relativamente nuevo, con una
comunidad en crecimiento y con bastante desarrollo últimamente. En general es fácil
de instalar y configurar aplicaciones (muy parecido a Rails) y tiene la belleza de unir todo
lo necesario para desarrollar una aplicación web en un solo lenguaje; JavaScript.
En nuestro grupo, hubo un consenso general sobre utilizar Node.js dado que varios
tienen experiencia y/o motivación por usar el framework. Al mismo tiempo, elegimos
MongoDB para la base de datos de la app, porque Node.js se integra fabulosamente
con MongoDB. Para Node.js existen muchas librerías open source, pero la que más se
destaca es Mongoose, una librería que modela los documentos de MongoDB a objetos de
JavaScript. Todas estas librerías están muy bien documentadas y casi todas se instalan
a través de npm (Node Package Manager).
